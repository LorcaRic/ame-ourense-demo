# Créditos de assets visuales

Este paquete incluye una colección ampliada de ilustraciones SVG originales creadas para la propuesta web de AME Ourense.

## Uso
- Las ilustraciones SVG creadas para este paquete pueden usarse y editarse dentro de esta web.
- El logotipo de AME procede de la imagen facilitada por el usuario y debe usarse solo si la asociación autoriza su uso.
- No se han incluido imágenes de bancos externos, por lo que no hay dependencia de licencias de terceros para las ilustraciones.

## Imágenes incluidas
- Cabeceras por página: historia, asociación, eventos, noticias, recursos y contacto.
- Tarjetas visuales para networking, formación, visibilidad, liderazgo, mentoría y alianzas.
- Ilustraciones editoriales para perfiles de junta/socias y noticias.
- Imagen 404, imagen de contacto y tiles decorativos.

## Sustitución futura
Cuando AME tenga fotografías reales de eventos, junta directiva o socias, se recomienda reemplazar algunas ilustraciones por fotografía profesional manteniendo la misma proporción: 16:10 en tarjetas, 21:8 en cabeceras de escritorio y 4:3 en móvil.
